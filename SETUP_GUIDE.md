# دليل نشر موقع تهنئة يوم المرآة على GitHub

## الخطوات:

### 1️⃣ إنشاء مستودع على GitHub
- اذهب إلى [github.com/new](https://github.com/new)
- اسم المستودع: `womens-day-site`
- اختر: `Public`
- اضغط: `Create repository`
- انسخ الرابط الذي سيظهر

### 2️⃣ افتح GitHub Codespace
- اضغط الزر الأخضر `Code`
- اختر `Codespaces`
- اضغط `Create codespace on main`

### 3️⃣ حمّل الملفات
في Terminal اكتب:
```
git config --global user.name "Your Name"
git config --global user.email "your-email@example.com"
```

### 4️⃣ أضف الملفات
```
git add .
git commit -m "Women's Day site"
git push
```

### 5️⃣ فعّل GitHub Pages
- اذهب إلى `Settings`
- اختر `Pages`
- اختر `main` و `/root`
- اضغط `Save`

### 6️⃣ انتظر 2-3 دقائق
الموقع سيكون متاح على:
```
https://your-username.github.io/womens-day-site/
```
